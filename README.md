# EEngine3D
EEngine3D is a 3D game engine created using OpenTK wrapper for OpenGL.
## Development Requirements

- OpenTK
- .NET 5.0


## Download / Clone

Windows, linux & OSX 

```sh
git clone https://github.com/ReformedMrozu/EEngine3D.git
```

or you can just download zip file of repo.
